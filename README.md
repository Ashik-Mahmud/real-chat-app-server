# Real Chat App - Server

This is real chat app using Node.js, Express, Socket.io, MongoDB, Mongoose, and React.

## Installation

```bash
npm install
```

## Usage

```bash

# Run in dev mode
npm run dev

# Run in prod mode
npm start
```
